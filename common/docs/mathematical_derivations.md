# 数学推导详述

## 四维拉普拉斯方程的推导

从三维薛定谔方程出发：
```math
(-½Δ - 1/r)Ψ = EΨ
```

通过变量替换和解析延拓，引入第四维坐标τ：

```math
Ψ(𝐱) → Ψ(𝐱,τ), τ = -it
```

得到四维拉普拉斯方程：

```math
Δ⁽⁴⁾Ψ(𝐱,τ) = (∂²/∂x² + ∂²/∂y² + ∂²/∂z² + ∂²/∂τ²)Ψ = 0
```

调和张量的显式构造

基于论文公式(31)，n阶调和张量的递归构造：

```math
M⁽ⁿ⁾ = (2n)!! 𝐲^⊗n - (2n-2)!!/(1!·2) ρ² Δ𝐲^⊗n + ...
```

其中收敛性由Lyapunov函数保证。
